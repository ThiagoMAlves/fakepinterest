FakePinterest
Um clone simplificado do Pinterest desenvolvido com Flask, SQLAlchemy, HTML, CSS e Bootstrap, com foco em aprendizado prático de desenvolvimento web fullstack com Python.
Projeto em desenvolvimento local. 

Como Rodar o Projeto Localmente:
Clone o repositório
git clone https://github.com/ThiagoMAlves/fakepinterest.git
cd fakepinterest

Instale as dependências
pip install -r requirements.txt

Execute a aplicação
python main.py

Sobre o Projeto
O FakePinterest é uma aplicação web inspirada no Pinterest, onde usuários podem:

- Criar contas
- Fazer login
- Criar perfis personalizados
- Enviar imagens
- Visualizar imagens de outros usuários
- Gerenciar suas próprias publicações

Este projeto foi criado com o objetivo principal de praticar desenvolvimento backend com Flask, integração com banco de dados, autenticação de usuários e templates dinâmicos com Jinja2.

Tecnologias Utilizadas
- Python
- Flask
- Flask-Login
- Flask-WTF
- Flask-SQLAlchemy
- SQLite
- HTML5
- CSS3
- Bootstrap
